# LittleRedsForestEscape

LittleRedsForestEscape is a 3D adventure game inspired by Little Red Riding Hood.

## Game Concept
The player controls Little Red Riding Hood and tries to escape through a dangerous forest while collecting crystals and avoiding obstacles.

## Features
- Crystal collection system
- Level progression
- Obstacles and traps
- Health system
- Game over / level complete screens

## Technologies Used
- Unity
- C#
- Mixamo animations
- Free 3D assets
